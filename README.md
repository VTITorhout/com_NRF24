# Detail

Code om communicatie op te zetten a.d.h.v. NRF24 (of equivalente) gebasseerde modules, dit geoptimaliseeerd voor de Arduino omgeving. Voorbeelden opgenomen voor o.a.:
* Uno (ATMEGA328P)
* ESP8266
* ESP32

De uitleg (en proefopstellingen) zijn opgedeeld in twee stukken, namelijk _unidirectionele_ en _bidirectionele_ communicatie.
Bij de bidirectionele communicatie wordt gebruik gemaakt van *acknowledging* om een *connection-oriented* verbinding te bekomen,
terwijl de unidirectionele communicatie eerder een *connectionless* verbinding is (zelfs indien er in beide richtingen communicatie wordt opgezet).
